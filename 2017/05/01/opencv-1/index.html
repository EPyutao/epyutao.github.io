Install

安装opencv for c++ winodws ：
为项目的属性管理器中添加库目录和包含目录，linker添加依赖项，并添加系统变量。
安装opencv for python，推荐使用conda管理包: 


conda install -c https://conda.binstar.org/menpo opencv3

基本类与方法

  • Mat：使用Mat数据结构来表示图像，能够自动进行内存管理。Mat类也重新定义了<<操作符，可以不经过for循环直接采用cout<<mat 来输出所有的像素值。该类提供了多种构造函数。对于多通道图像而言，矩阵的列中包含多个子列，其子列个数与通道数相等。mat.size()获得图像大小。
  • at函数：实现读Mat数据结构中的某个像素，或者对某个像素进行复制操作。
    
uchar value = mat.at(i, j); //读取i,j处的像素值
mat.at(i, j) = 128; //将i,j处的像素值设为128
  • MatIterator迭代器：高效进行矩阵元素的遍历。

    MatIterator_ begin, end;
    for(begin = img.begin(), end = img.end(); begin!=end; ++begin)
    {
    *begin = TODO
    }
  • 指针遍历： uchar *p = mat.ptr(i) //获取第i行的像素指针
  • filter2D函数：滤波器掩码核函数，参数为输入图像Mat, 输出图像，输入图像通道数，以及用到的核函数。

    //定义核函数
    Mat kern = (Mat_(3,3)<< 0, -1, 0,-1, 5, -1, 0, -1, 0);
    //调用filter2D函数
    filter2D(Input, Output, Input.depth， kern)

    调整kern矩阵，还可以以邻域算子方式改变图像亮度
  • addWeighted函数：对两幅图像进行blending混合。参数为输入图像mat1, 权重alpha1, 输入图像2， 权重alpha2， 偏置beta， 输出图像mat3
  • dft():离散傅里叶变化函数。
  • 通道转换:cvtColor(src, dst, ColorConversionCodes)：将图像的颜色空间转换为另一种模式。

基本绘图

  • point:point数据结构表示了一个2D的点，定义为： Point pt = Point(10, 8);
  • Scalar：该数据结构表示了具有4个元素的数组。在opencv中用于传递像素值: Scalar(blue, green, red);
  • Myline(Mat img, Point start, Point end):绘制一条从start到end的直线段。
  • MyEllipse(Mat img, double angle)：绘制一个椭圆，椭圆旋转角为angle
  • MyFilledCircle(Mat img, Point center):绘制实心椭圆。
  • MyPolygon(Mat img)：绘制多边形。以上方法绘制的图像大小和中心为宽度的固定百分比。

imgproc模块

  • 平滑处理：blur(src, dst, size(W,H), Point(-1,-1))：kern大小，point被平滑点位置； GaussianBlur(src, dst, size(W,H),theta1, theta2):kern大小，theta为x,y方向标准差; medianBlur(src, dst, i):i为内核大小，必须为奇数,中值滤波器; bilateralFilter(src, dst, d, thetaColor, thetaSpace),双边滤波器。
  • 腐蚀与膨胀(Eroding and Dilating):将图像与内核进行卷积操作，将卷积核内区域最大值代替锚点(使亮区拓展)，或最小值代替锚点(使暗区拓展).erode与dilate函数均接受输入mat,输出mat,kern这三个参数，kern可以为Mat格式数据结构，可以有矩形(MORPH_RECT)，交叉形(MORPH_CROSS)，椭圆形(MORPH_ELLIPSE)等形状卷积核。
    
Mat element = getStructuringElement(type, Size, Point);
erode(src, dst, element);
    先进行erode再进行dilate，即对图像进行开运算(opening)，能排除小型亮色区域；新进行dilate再进行erode，即对图像进行闭运算(closing)，能排除小型暗色区域。形态梯度(Morphological Gradient)：dilate - erode；顶帽(Top Hat): src - open(src); 黑帽(balck hat): close(src) - src。以上五种形态学运算可以采用morphologyEX来操作 
    
Mat ele = getStructuringElement();
morphologyEX(src, dst, Morphology_Operations, ele);
  • 图像金字塔: 1.Gaussian pyramid高斯金字塔：图像与Gussian kern卷积，再讲所有的偶数行与列去除，是一个向下采样的方式。2.Laplace pyramid拉普拉斯金字塔:高斯金字塔丢失的数据构成了laplace金字塔，用来向上重建图像。
    
pyrDown(src, dst, Size(src.cols/2, src.rows/2))
pyrUp(src, dst, Size(src.cols*2, src.rows*2))
  • 阈值操作:threshold(src, dst, threshold_value, max_grayvalue, threshold_type)接受的参数为输入输出图像，阈值，最大灰度值，阈值的类型：0: 二进制阈值 1: 反二进制阈值 2: 截断阈值 3: 0阈值 4: 反0阈值
  • 添加边界:copyMakeBorder(src, dst, topWidth, bottomWidth, leftWidth, rightWidth, borderType, value)接受参数，boderType接受两个参数：BORDER_CONSTANT；BORDER_REPLICATE。
  • Sobel导数：Sobel是一个离散的微分算子，用来计算图像灰度函数的近似梯度。
    
//Sobel(src_gray, grad_x, ddepth = CV_16S, x_order, cale, delta, BORDER_DEFAULT)
//求x方向导数
Sobel( src_gray, grad_x, ddepth, 1, 0, 3, scale, delta, BORDER_DEFAULT )
//将两个方向的绝对值加和便获得梯度值。
grad = abs(grad_x) + abs(grad_y)
  • Laplacian算子：一个离散的微分算子，用来求二阶导数。不同于sobel函数，Laplacian直接用来边缘检测。
    
Laplacian( src_gray, dst, ddepth, kernel_size, scale, delta, BORDER_DEFAULT );
  • Canny边缘检测：最优边缘检测方法：1.采用GaussianBlur进行卷积降噪; 2.用sobel算子计算一阶导数，并计算梯度方向角，而且将梯度方向角近似到(0,45,90,135)四个可能角度中; 3.非极大值抑制，排除了非边缘像素; 4.滞后阈值，保留高度值处的值，剔除低阈值处的值，处于中间的值则仅当其连接到一个高阈值时保留。

    Canny( src, dst, lowThreshold, heighThreshold, kernel_size );

    输出结果为0和255两个值。
  • HoughLine霍夫线变化：霍夫线变化是一种用来寻找直线的方法。在用霍夫线变化之前，要先对图像进行边缘检测处理，即霍夫线变化的直接输入只能是边缘二值图像。OpenCV实现了以下两种霍夫线变换：标准霍夫线变换，它能给我们提供一组参数对的集合来表示检测到的直线，在OpenCV中通过函数HoughLines来实现；统计概率霍夫线变换，这是执行起来效率更高的霍夫线变换. 它输出检测到的直线的端点在OpenCV 中它通过函数HoughLinesP来实现。前者返回vector类型的数据，后者返回vector类型数据。
  • HoughCircles霍夫圆变化：霍夫圆变换的基本原理与霍夫线变换类似, 只是点对应的二维极径极角空间被三维的圆心点x, y还有半径r空间取代.

    vector circles;
    HoughCircles( src_gray, circles, CV_HOUGH_GRADIENT, dp=1, min_dist = src_gray.rows/8, param_1=200, param_2=100, min_radius=0, max_radius=0 );

    返回一个vec3f的向量，前两位为圆心坐标，最后一位为半径。接受的变量为：1.srcgray: 输入图像 (灰度图)；2.circles: 存储下面三个参数: x{c}, y_{c}, r 集合的容器来表示每个检测到的圆； 3.CV_HOUGH_GRADIENT: 指定检测方法. 现在OpenCV中只有霍夫梯度法； 4.dp = 1: 累加器图像的反比分辨率； 4.min_dist = src_gray.rows/8: 检测到圆心之间的最小距离； 5.param_1 = 200: Canny边缘函数的高阈值； 6.param_2 = 100: 圆心检测阈值； 7.min_radius = 0: 能检测到的最小圆半径, 默认为0； 8.max_radius = 0: 能检测到的最大圆半径, 默认为0
  • resize尺度变化: resize(src, dst, dsize, fx, fy, interpolation)函数，用于变换图片尺寸，dsize接受一个Size(width, heigh)的参数，fx,fy接受一个x和y方向的缩放和拉伸比例。
  • Remapping重映射：即把一个图像中一个位置的像素放置到另一个图片指定位置的过程. remap(src, dst, InputArray-map_x, InputArray-map_y, interpolation, borderMode, Scalar()) ，在处理中，将map_x.at(j, i) = TODO; 作为映射方法函数在x方向的处理函数。
  • Affine Transformation仿射变换:任意一个仿射变化都可以表示为图像mat乘以一个矩阵(线性变换)加上一个向量(平移);在opencv中采用一个2x3矩阵来表示仿射变换。仿射变换有两个主要api，一个是通过目标图像与源图像联系，获得两图像之间的仿射变换矩阵(即2x3矩阵) AffineMatrix = getAffineTransform( srcList, dstList), AffineMatrix数据结构为Mat型; 另一个是调用仿射变换矩阵，应用到图像上:warpAffine(src, dst, AffineMatrix, dst.size()),其中dst.size()是Size类型数据。而旋转仿射变换矩阵可以通过getRotationMatrix2D(center, angle, scale)来获得：其中center为Point类型数据结构，angle为旋转角度(顺时针)，scale为缩放因子。
  • Histogram直方图相关: 直方图是图像中像素强度分布的图形表达方式. 1.直方图均衡化:将源图的直方图分布映射到一个归一化的累积分布函数cdf, equalizeHist(src, dst)将原图均衡化,注意此处src接受一个灰度图为参数; 2.计算直方图:calcHist( &src, nimages, channels, InputMask, OutputArray, dims, histSize, Ranges, uniform, accumulate): src要传入指针&src, OutputArray的数据结构类型是Mat, dims:直方图的维数,histSize:bin数目采用指针形式，Ranges:数据为二维数组,每个维度的取值范围Ranges(0,0)~Ranges(0,1); 3.直方图对比：compareHist(hist1, hist2, compareModel)
    //计算与绘制直方图:
cvtColor(src, src, CV_BGR2GRAY);
int histSize = 256;
int hist_size[] = { histSize };
float range[] = { 0, 256 };
cout << range[1]<< endl;
const float* ranges[] = { range };
cout << ranges[0][1];
Mat hist;
calcHist(&src, 1, 0, Mat(), hist, 1, hist_size, ranges, true, false);
cout << "calcHist Size: " << hist.size() << endl;
//绘制画布
int hist_w = 400;
int hist_h = 400;
int bin_w = cvRound((double)hist_w / histSize);//地板除,获取bin的宽度
cout << bin_w;
Mat histImage(hist_w, hist_h, CV_8UC3, Scalar(0, 0, 0));
// 归一化直方图
normalize(hist, hist, 0, histImage.rows, NORM_MINMAX, -1, Mat());
// 直方图画布绘制直方图
for (int i = 1; i < histSize; i++) {
line(histImage, Point(bin_w*(i-1), hist_h - cvRound(hist.at(i - 1))), Point(bin_w*(i), hist_h - cvRound(hist.at(i))), Scalar(0, 0, 255), 2, 8, 0);
}
namedWindow("calcHist Demo", CV_WINDOW_AUTOSIZE);
imshow("calcHist Demo", histImage);
  • 反向投影:计算某一特征的直方图，然后使用模型去寻找图像中存在的该特征，类似于HOG行人检测.调用的API为: calcBackProject(&src, nimages, &channels, Hist, OutputArray, &ranges, scale, uniform)，通过调节Hist，产生不同特征的mask来对src进行反向投影。
  • 模板匹配:在一幅图像中寻找与另一幅模板图像最匹配的技术。模板匹配操作: matchTemplate(img, templ, result, match_method); result是结果匹配矩阵， 使用minMaxLoc来确定矩阵result中最大最小值的位置。minMaxLoc( result, &minVal, &maxVal, &minLoc, &maxLoc, Mat() ); 其中minLoc与maxLoc均为Point数据类型。
  • 凸包: findContours用来寻找轮廓,返回一个contours(数据结构vector>)，contours.size()表明了有多少个轮廓，contours[i]则表明了该轮廓点构成; drawContours用来绘制轮廓接受一个contours; contours[i]单独表示一个点构成，approxPolyDP，boundingRect，minEnclosingCircle，minAreaRect，fitEllipse均可以接受这类型的参数，并返回该点构成的多边形逼近,矩形边界框,圆形边界框,最小包围矩形，椭圆。boundingRect计算出数值类型为vector 。对于每个contours[i]数据而言，都可以 使用contourArea 来计算轮廓面积， arcLength 来计算轮廓或曲线长度。
